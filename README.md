# Date Night Games

A real-time multiplayer web platform for 2-player games. Connect with someone on a different device and play together.

## Features

- Real-time WebSocket connections
- Room-based matchmaking
- Multiple game options
- Responsive design

## Tech Stack

- **Frontend:** React + Vite + TypeScript
- **Backend:** Node.js + Express + Socket.io
- **Styling:** Tailwind CSS

## Games

### Launch Titles
- Connect 4
- Chess
- Memory Match
- Word Scramble
- Pong

### Planned
- Checkers
- Battleship
- Trivia Duel
- And more...

## Getting Started

```bash
# Install dependencies
npm install

# Run development server
npm run dev
```
